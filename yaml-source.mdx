# Source Content from YAML Within Gatsby

Set up Gatsby to source data from YAML filea directly into a page or component.

This recipe installs and configures all the necessary NPM packages and Gatsby plugins to enable sourcing of content from YAML files.

---

## Install NPM Packages

<NPMPackage name="gatsby-source-filesystem" />

<NPMPackage name="gatsby-transformer-yaml" />

---

# Install Gatsby Plugins

First `gatsby-source-filesystem` to point to our YAML files which we'll keep in the `/src/data` directory.

<GatsbyPlugin
  name="gatsby-source-filesystem"
  key="src/data"
  options={{
    path: `./src/data/`,
  }}
/>

---

Then the `gatsby-transformer-yaml` plugin.

<GatsbyPlugin name="gatsby-transformer-yaml" />

---
